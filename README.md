# Tarefando
Tarefando is a web application developed with a focus on simple task management, based on task types with colors.

## Setup
Node.js and npm are required to run this project. You can download them from [nodejs.org](https://nodejs.org/).

Install dependencies:
```bash
npm install
```

Run the application:
```bash
npm run dev
```

---
Check out the project's backend API [here](https://github.com/drmcarvalho/Tarefando).
